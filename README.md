# git branch
- git branch branch_name : 브랜치 생성
- git branch : 브랜치 목록보기
- git checkout branch_namme  : 브랜치 선택
- git branch -m branch_name change_branch_name : 브랜치 이름 바꾸기
- git branch -d branch_name : 브랜치 삭제하기
